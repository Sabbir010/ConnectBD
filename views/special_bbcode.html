Special BBCode List

Here is a list of all available BBCodes for premium members. You can use these codes in shouts, topic posts, archive posts, and replies to format your text and create shortcuts.

BBCode Example Description
Text Formatting
[b]TEXT[/b] Bold Text Makes text bold.
[u]TEXT[/u] Underlined Text Underlines text.
[i]TEXT[/i] Italic Text Makes text italic.
[big]TEXT[/big] Big Text Makes text larger.
[small]TEXT[/small] Small Text Makes text smaller.
[clr=red]TEXT[/clr] Red Text Changes text color. You can use color names (e.g., blue, green) or hex codes (e.g., #FF5733).
[br/] Line
Break		 Adds a line break.
Dynamic Content & Links
[t=topic_id] [t=123] Creates a link to a topic with its title. Replace '123' with the actual Topic ID.
[s=shout_id] [s=456] Creates a link to a specific shout. Replace '456' with the Shout ID.
[a=archive_id] [a=789] Creates a link to an archive with its title. Replace '789' with the Archive ID.
@username@ @ConnectBD@ Tags a user, sending them a notification.
[user] Hello [user] Displays the name of the user who is currently viewing the text.
[youtube=video_id]Your Text[/youtube] [youtube=dQw4w9WgXcQ]Best Video[/youtube] Embeds a YouTube video. Replace 'dQw4w9WgXcQ' with the video ID from the URL.
